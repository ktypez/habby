# Habby

Track your habits with gamification and streaks.

## Features

- **Streak Tracking**: Daily habit streaks with fire icons
- **Progress Visualization**: Calendar heatmap and weekly views
- **Level Progression**: XP-based leveling system
- **Achievements**: Unlock achievements for milestones
- **Analytics**: Detailed statistics and insights

## Installation

```bash
npm install
npm run dev
```

## Quick Start

1. Track daily habits with streak counters
2. Watch your level progress with XP bars
3. View your completion rates in analytics
4. Unlock achievements as you build consistency

## Development

- View the app: http://localhost:3000
- Build for production: `npm run build`
- Preview built app: `npm run preview`

## Technologies

- Astro 7 (SSR)
- Alpine.js 3
- TypeScript
- Tailwind CSS (via utility classes)
- Reactively state management with Jotai

Design inspired by **Trophy UI Kit**
